# AI-Based Health Prediction System

## Overview

This project predicts a person's health condition using Machine Learning based on:

- Body Temperature
- Pulse Rate

The system classifies health conditions into:

- Healthy
- Moderate
- Serious

A Random Forest Classification model is trained on a healthcare dataset and integrated with a Python Tkinter GUI application.



## Features

- AI-based health prediction
- User-friendly GUI
- Input validation
- Real-time prediction
- Healthcare recommendations
- Approximately 95% prediction accuracy



## Technologies Used

- Python
- Scikit-Learn
- Pandas
- NumPy
- Joblib

## How to Run

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Train Model

```bash
python train_model.py
```

### Run Application

```bash
python gui_app.py
```
